## 🎮 How to Play

### 🚀 **Getting Started**
1. **Enter your Minecraft name** (required)
2. **Click "Challenge for the Crown!"** 
3. **Watch the cinematic intro** (or skip)
4. **Start your mission!**

### 🎯 **Gameplay Mechanics**
- **Arrow controls** - Move characters with arrow keys or on-screen buttons
- **Germ avoidance** - Red germs (-10 points) and Green germs (-5 points)
- **Character switching** - First Maitha, then Jamilah
- **Score management** - Start with 100 points, avoid germs!

### 🏆 **Scoring System**
- **Red Germs** 💀 = **-10 points** (Dramatic red glow effect)
- **Green Germs** 🦠 = **-5 points** (Dramatic green glow effect)
- **Hall of Fame** 🏆 = **Live leaderboard ranking**
